# Study-Time-Tracker
A tool for tracking time spent studying. Users can track multiple sessions per subject, and keep track of older sessions via persistent memory.

## Motivation

Before starting the next part of my education in Computer Science, I wanted to improve my time-management skills by tracking how I spent my time studying. Seeing as all the available time-trackers online are either behind a paywall or lack the features I desire, this provided me the perfect opportunity to sharpen my web-development skills.
